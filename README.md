 # Trader Behavior vs. Market Sentiment Analysis

### Project Overview
An investigation into how Bitcoin market sentiment (Fear & Greed Index) correlates with actual trader performance and risk-taking on the Hyperliquid exchange.

### Key Insights
- **High Conviction in Greed:** Traders show 10x higher PnL efficiency during 'Extreme Greed' compared to 'Extreme Fear'.
- **The Retail Trap:** Win rates drop by 13% during high volatility 'Fear' phases.
- **Top Trader Edge:** The top 5% of earners accumulate positions earlier in 'Neutral' phases while retail waits for 'Greed'.
